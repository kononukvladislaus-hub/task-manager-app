# 📋 Task Manager - Менеджер задач

## 🎯 Цель проекта
Создание простого и удобного веб-приложения для управления личными задачами и повышения продуктивности.

## 📝 Описание
Task Manager - это полнофункциональное веб-приложение, которое позволяет:
- Создавать новые задачи
- Отмечать выполненные задачи
- Редактировать существующие задачи
- Удалять ненужные задачи
- Фильтровать задачи по статусу

## 🛠 Технологический стек

### Backend:
- **Python 3.8+**
- **Flask** - легкий веб-фреймворк
- **SQLite** - база данных

### Frontend:
- **HTML5** - структура страниц
- **CSS3** - стилизация и адаптивный дизайн
- **JavaScript (Vanilla)** - интерактивность

## 🏗 Архитектура приложения
### Структура проекта:
- `app.py` - основной файл приложения, маршруты и логика
- `templates/index.html` - главный HTML-шаблон
- `static/style.css` - стили приложения
- `static/script.js` - клиентская логика

## 📋 Функциональные требования

### Реализовано:
- ✅ Создание задачи с названием и описанием
- ✅ Отметка о выполнении (checkbox)
- ✅ Редактирование задачи
- ✅ Удаление задачи
- ✅ Фильтрация (все/активные/выполненные)
- ✅ Счетчик задач
- ✅ Адаптивный дизайн

### Планируется (v2.0):
- 🔲 Приоритеты задач (высокий/средний/низкий)
- 🔲 Дедлайны и напоминания
- 🔲 Категории и теги
- 🔲 Поиск по задачам
- 🔲 Экспорт/импорт данных
